## Bilibili 用户视频插件

获取指定 B站 UP主的最新视频投稿，在时间线上显示视频信息。

### 功能特性

- 实时获取UP主最新视频投稿
- 显示视频标题、播放量、弹幕数、发布时间
- 智能30分钟缓存，减少API请求
- 点击事件可直接跳转到B站视频页面
- 视频封面作为事件图标

### 配置参数

#### 必需参数

| 参数名     | 类型     | 说明         | 示例      |
|---------|--------|------------|---------|
| mid | string | B站用户ID（必需） | 2826561 |

#### 可选参数

| 参数名      | 类型     | 默认值  | 说明              |
|----------|--------|------|-----------------|
| pageSize | string | "10" | 每次获取的视频数量（1-50） |

### 如何获取用户ID（mid）

1. 从个人主页URL获取
    - 访问UP主的个人主页
    - URL格式：https://space.bilibili.com/2826561/
    - 数字部分即为用户ID：2826561
2. 从视频页面获取
    - 打开UP主的任意视频
    - 点击UP主头像进入个人主页
    - 查看URL中的数字部分
